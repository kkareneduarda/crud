## Script CRUD Básico em PHP

## Descrição
Este projeto é um CRUD simples em PHP, sem uso de banco de dados, que manipula os dados de um cadastro de usuários.

## Tecnologias Usadas
- PHP
- HTML básico

## Funcionalidades

- Criar um registro
- Listar registros
- Atualizar um registro
- Deletar um registro

## Estrutura do Projeto

crud/
│
├── index.php     # Menu principal
├── dados.php     # Simula um banco de dados
├── criar.php     # Criar usuário
├── listar.php    # Listar usuários
├── atualizar.php # Atualizar usuário
└── deletar.php   # Deletar usuário



##  Executando o projeto
1. Clone o repositório
2. Coloque o projeto no htdocs
3. Inicie o XAMPP
4. Acesse `http://localhost/crud/index.php`
